# BGM License Information

Document the license information for each BGM track used in Pashabook.

## Template

For each BGM file, add an entry following this format:

```
[filename].mp3
- Track: "[Track Name]" by [Artist Name]
- License: [License Type] (e.g., CC0, CC BY 4.0, Royalty-Free)
- Source: [URL where track was obtained]
- Attribution: [Required attribution text, if any]
- Date Added: [YYYY-MM-DD]
```

## License Compliance Notes

- **CC0 (Public Domain)**: No attribution required, free for commercial use
- **CC BY 4.0**: Attribution required, free for commercial use
- **Royalty-Free**: Check specific terms, usually requires one-time purchase

## Important Reminders

1. Always verify license terms before using any track
2. Keep attribution text if required by the license
3. Update this file whenever BGM tracks are changed
4. For production, consider purchasing commercial licenses for higher quality
